# jobsity-chat
## Description

Mono repo for the Jobsity chat. You can check the separated repos on the following links:

* https://github.com/dacm26/nestjs-chat-server
* https://github.com/dacm26/angular-chat-client

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Bootstrap the repo
$ npm run boostrap

# start the project
$ npm run start
```
